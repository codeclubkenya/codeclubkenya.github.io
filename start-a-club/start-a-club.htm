A worldwide network of coding clubs for primary school pupils.

Start a club

Code Club is just getting started in Kenya right now. If you wish to get involved, or are ready to get going and want to register a new club please use the Contact Us form so we can support you in your journey.

Code Club is all about connecting talented computer programmers with their local schools and providing them with the materials they need to help kids learn to code.
